Acaba de salir la nueva versión de Angular, la versión 8 de este gran framework para JavaScript. Esta nueva versión no representa un cambio radical con respecto a la versión 7, es decir, todo el código que tengamos hecho con Angular 7 es totalmente compatible con la versión 8, no hay cambios de sintaxis, por lo cual la migración de una versión a otra será muy fácil y no tendrás ningún tipo de problema..
